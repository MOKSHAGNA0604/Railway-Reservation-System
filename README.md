# Railway-Reservation-System

Project Overview
The **Railway Reservation System** is a Java-based application designed to simplify the process of booking railway tickets online.  
It allows users to register, log in, view available trains, book tickets, and manage reservations efficiently.

This project is suitable for **academic purposes** and helps understand **Java, Servlets, HTML, and CSS integration**.

**Technologies Used
- Java (Core Java & Servlets)
- HTML
- CSS
- Apache Tomcat Server
- Eclipse IDE
- Git & GitHub

  **Features
- User Registration
- User Login
- Forgot Password option
- View Train Details
- Ticket Booking
- Simple and user-friendly interface
